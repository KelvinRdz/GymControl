# GymControl

GymControl es un sistema web universitario para administrar clientes, entrenadores, membresías, pagos, rutinas y asistencias de un gimnasio.
link del video y zip de avance:
https://drive.google.com/drive/folders/1KKEoJyK1HXeI6QWEsvjUVvOuqhxsKSCb?usp=sharing

## Tecnologías
- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- Thymeleaf
- Bootstrap 5
- MySQL
- Maven
- NetBeans

## Requisitos
- Java 21
- Maven
- MySQL

## Ejecución
1. Crear la base de datos llamada `gymcontrol` en MySQL.
2. Ajustar las credenciales en `src/main/resources/application.properties`.
3. Ejecutar el proyecto con NetBeans o con Maven:
   ```bash
   mvn spring-boot:run
   ```
4. Abrir en el navegador:
   ```text
   http://localhost:8080
   ```
